# Tiny crm

This is a app created for Linode hackathon on [Dev](https://dev.to/devteam/announcing-the-linode-dev-hackathon-377p).

## Tech stack

- PHP (Laravel)
- Filament PHP
- Tailwind CSS
- Alpine JS
- PostgreSQL
